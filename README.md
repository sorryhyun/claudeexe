# Claude Mascot

A desktop mascot app featuring Clawd, built with Tauri v2 + React + TypeScript.

## Features

- Transparent, always-on-top window
- Draggable character
- System tray support

## Commands

```bash
npm install      # Install dependencies
npm run dev      # Run in development mode
npm run build    # Build for production
npm run icons    # Regenerate icons from source image
```

## Tech Stack

- Tauri v2
- React + TypeScript
- Vite
